# Archivos obsoletos

Los siguientes archivos **no deben usarse** en la entrega actual:

| Archivo | Sustituto |
|---------|-----------|
| `web_app.py` | `main.py` + `server.py` (Flask) |
| `src/ui.py` | Interfaz web en `templates/` + `static/` |

Ejecute siempre:

```powershell
cd "APP hecha"
run.bat
```

o `py -3 main.py` → http://127.0.0.1:5000
