Cómo funciona (para usted)

Use Ruta aleatoria para una demo o escriba el estante y el muelle. Verá el stock y la ubicación.

Sobre SmartRoute WMS

Navegación inteligente dentro de su centro de distribución

Imagínese Google Maps, pero para un almacén de tecnología de 120×80 metros: usted marca dónde recoger mercancía y dónde entregarla, y el sistema traza la ruta evitando pasillos congestionados en hora pico.

No necesita saber de algoritmos. La aplicación le muestra el mapa interactivo (zoom, clic en nodos), el stock en el estante, el robot más adecuado y una explicación clara: ¿por qué esta ruta sí y la directa no?

CÓMO EJECUTAR

  1. Abra una terminal en la carpeta APP hecha
  2. Ejecute run.bat o py -3 main.py
  3. Abra http://127.0.0.1:5000 (no abra el HTML directamente)
  4. Si el puerto 5000 está ocupado: set PORT=5055 y vuelva a ejecutar
